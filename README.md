# 远程调用手机摄像头

通过 WebSocket 实现手机摄像头画面的实时传输，在电脑端进行查看。支持使用皎月连进行内网穿透远程访问。

## 项目架构

```
┌─────────────┐     WebSocket      ┌─────────────┐     WebSocket      ┌─────────────┐
│  📱 手机端   │ ──→ 视频帧 ──→    │  🔌 服务器    │ ──→ 视频帧 ──→    │  🖥️ 电脑端   │
│  camera.html│                    │    app.py    │                    │ viewer.html │
└─────────────┘                    └─────────────┘                    └─────────────┘
```

## 技术栈

- **后端**: Python 3 + Flask + Flask-SocketIO
- **前端**: HTML5 + CSS3 + JavaScript
- **通信**: WebSocket (Socket.IO)
- **视频采集**: MediaDevices API (`getUserMedia`)
- **安全**: 自签名 SSL 证书（浏览器要求 HTTPS 才能访问摄像头）
- **远程访问**: 皎月连（内网穿透）

## 运行流程

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 选择启动方式

根据你的网络环境选择不同的启动文件：

| 启动文件 | IP 方式 | 适用场景 |
|----------|---------|----------|
| `app.py` | 固定 IP (`10.6.22.1`) | 使用皎月连等内网穿透工具 |
| `app_autoip.py` | 自动获取局域网 IP | 手机和电脑在同一局域网 |

#### 局域网模式（推荐）

```bash
python app_autoip.py
```

程序会自动检测你的电脑局域网 IP，无需手动配置。启动后会显示类似：

```
  检测到本机IP: 192.168.1.100
  手机端打开: https://192.168.1.100:5000/camera
  电脑端打开: https://192.168.1.100:5000/viewer
```

#### 皎月连模式（远程访问）

```bash
python app.py
```

使用固定 IP `10.6.22.1`，适用于通过皎月连进行远程访问。

服务器会自动生成 SSL 证书（首次运行时），然后在端口 5000 启动服务。

### 3. 手机端操作

1. 在手机浏览器中打开：`https://<服务器IP>:5000/camera`
2. 浏览器提示证书不安全时，点击「高级」→「继续访问」
3. 输入摄像头名称（默认为"我的手机"）
4. 点击 📷 按钮开始推流
5. 可选操作：
   - 🔄 切换前后摄像头
   - 💡 开启/关闭闪光灯（需设备支持）
   - 调整画质（标清/高清/超清）

### 4. 电脑端查看

1. 在电脑浏览器中打开：`https://<服务器IP>:5000/viewer`
2. 等待手机端开始推流后，画面会自动显示
3. 查看实时统计：FPS、延迟、帧大小
4. 点击 ⛶ 按钮进入全屏模式

## 核心流程说明

### 连接流程

```
手机端                    服务器                    电脑端
  │                          │                          │
  │── SocketIO connect ─────→│                          │
  │                          │                          │
  │←── connect 确认 ─────────│                          │
  │                          │                          │
  │── register_camera ──────→│                          │
  │                          │── broadcast camera_list →│
  │                          │                          │
  │                          │←── register_viewer ──────│
  │                          │── camera_list (当前列表)→│
  │                          │                          │
```

### 视频帧传输流程

```
手机端                    服务器                    电脑端
  │                          │                          │
  │  getUserMedia 获取流      │                          │
  │         ↓                │                          │
  │  Canvas 捕获帧           │                          │
  │         ↓                │                          │
  │  toDataURL 转 JPEG       │                          │
  │         ↓                │                          │
  │── video_frame ──────────→│                          │
  │                          │── emit video_frame ─────→│
  │                          │       (room: viewer_sid) │
  │                          │                          │
  │                          │                          │── img.src = data.image
```

## 使用皎月连实现远程访问

如果手机和电脑不在同一局域网，可以使用 **皎月连** 内网穿透工具实现远程访问。

### 皎月连简介

皎月连是一款免费的内网穿透工具，支持组网模式和端口映射模式，可实现跨网络远程访问。

### 配置步骤

#### 服务端（运行 app.py 的电脑）

1. 下载并安装皎月连（支持 Windows、Linux、安卓）
2. 打开软件并登录，在左侧选择「服务端」
3. 开启「组网模式」开关
4. 设置页面访问密码（6-20位）
5. 点击「开启服务」按钮，成功后按钮变为黄色

#### 客户端（手机）

1. 在手机上下载并安装皎月连（安卓）
2. 在左上方列表中选择在线主机（服务端的主机名）
3. 点击下方「连接」按钮，连接成功后按钮变为黄色
4. 记住映射列表中的虚拟局域网 IP：**10.6.22.1**
5. 按手机 Home 键将皎月连后台运行
6. 打开浏览器访问：`https://10.6.22.1:5000/camera`

### 说明

- **虚拟局域网 IP**: 皎月连组网模式下，服务端的虚拟 IP 默认为 `10.6.22.1`
- **端口映射**: 确保服务端开启了端口映射，将 5000 端口映射出去
- **证书问题**: 首次访问仍会提示证书不安全，点击「高级」→「继续访问」

### 替代方案

如果使用端口映射模式：

1. 服务端关闭「组网模式」，添加端口映射：
   - 局域网 IP: `127.0.0.1`（或本机局域网 IP）
   - 本地端口: `5000`
   - 协议: TCP
2. 客户端连接后，使用 `https://127.0.0.1:<远程端口>/camera` 访问

## 注意事项

1. **HTTPS 必须**: 现代浏览器要求 HTTPS 才能访问摄像头 API
2. **网络要求**: 手机和电脑需在同一局域网内（或通过皎月连等内网穿透工具连接）
3. **证书警告**: 首次访问会提示证书不安全，需手动确认继续
4. **性能建议**: 在网络不稳定时建议使用标清画质
5. **浏览器兼容性**: 推荐使用 Chrome、Safari 或 Edge
